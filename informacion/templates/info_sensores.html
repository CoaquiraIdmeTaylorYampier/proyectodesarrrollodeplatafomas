INFORMACIÓN KIT SENSORES

Kit Completo de Sensores Arduino UNO R3

Este kit incluye una amplia variedad de sensores que te permitirán interactuar con el mundo físico y medir diferentes magnitudes, ideal para proyectos avanzados con Arduino.

  • Sensor de movimiento PIR
  • Sensor de temperatura y humedad (DHT11)
  • Sensor ultrasónico de distancia
  • Sensor de luz (fotorresistencia)
  • Sensor de sonido
  • Sensor de vibración
  • Módulo joystick
  • Sensor de nivel de agua
  • Módulo RFID
  • Y muchos más...

Amplía las capacidades de tus proyectos Arduino con esta colección versátil de sensores.
